VR

VR編集

一般名称編集

  • バーチャル・リアリティ (virtual reality) - 仮想現実。
  • 録画 (video recording)
  • ビデオ信号記録装置 (video recorder)
  • 電子部品
    • 可変抵抗器 (variable resistor)
    • 電圧レギュレータ (voltage regulator)
    • LEDの逆電圧のこと(LED reverse voltage)
  • 静脈還流 (venous return)
  • 取締役会 (Verwaltungsrat)
  • 付値環 (valuation ring)

固有名称編集

  • ゲーム
    • バーチャレーシング - セガのレースゲーム。
    • 電脳戦機バーチャロン - セガの対戦シューティングゲーム。
      • バーチャロイド - 上記に登場する架空のロボット。
  • 企業
    • ビデオリサーチ (Video Research) - 日本の視聴率調査会社。
    • 静岡放送 (JOVR / JOVR-DTV)
    • TACV (TACV、IATA: VR) - カーボベルデの航空会社。別名 カーボベルデ航空。
    • VRグループ - フィンランドの国有鉄道会社。
    • バチカン放送 (Vatican Radio)
    • ヴィクトリア鉄道 (Victorian Railways) - オーストラリアの鉄道会社。
    • ブイアール - テイチクエンタテイメントと株式会社アズミックスが立ち上げた音楽レーベル。
  • 規格
    • DVD-VR（VRモード）
    • QuickTime VR (QuickTime VR) - 画像ファイルフォーマット。
  • 音楽バンド
    • ヴェルヴェット・リヴォルヴァー (Velvet Revolver) - アメリカ合衆国のロックバンド。
    • ヴェンジャンス・ライジング (Vengeance Rising) - アメリカ合衆国のメタルバンド。
  • ヴェローナ県 (Verona、ISO: VR) - イタリアの県。
  • HiBOOM VR - エイベックス系の声優専門レーベル。
  • ヴィクトリア女王 (Victoria Regina)。
  • 手ぶれ軽減 (Vibration Reduction) - ニコンのカメラ技術。

VR編集

  • V速度（航空機離陸時の速度基準）の1つ rotation speed。
